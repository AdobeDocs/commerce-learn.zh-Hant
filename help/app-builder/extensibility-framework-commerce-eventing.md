# App Builder的擴展框架

要瞭解如何將Adobe Commerce與Adobe DeveloperApp Builder配合使用，請查看以下視頻，其中顯示如何與App Builder一起使用Commerce事件。 此功能演示描述了整合的工作原理並探索了App Builder的潛力。 本演示介紹了可擴充性框架Commerce Eventing，以及如何使用它來開發AmazonSales Channel擴展。
