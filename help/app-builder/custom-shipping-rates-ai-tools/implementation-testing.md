# AI工具的自訂運費 — 實作與測試

瞭解如何實作及測試AI驅動的自訂運費。 本教學課程會引導您完成環境設定、程式碼清理和擴充功能部署。 您也將探索測試程式，包括電信業者註冊、設定更新和店面驗證，以確保運費整合的準確性。
