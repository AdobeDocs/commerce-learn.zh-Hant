# 建立產品屬性

Adobe Commerce使用屬性來管理與產品、客戶和產品評級相關的資訊。 屬性是一些資訊，可能在表單、報告、產品評論、分層導航和電子郵件中顯示為欄位。 它們是產品目錄的構件，並描述產品的特定特徵。

這段錄像是給誰的？

- 網站經理
- 電子商務商品商
