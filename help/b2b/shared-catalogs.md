# 管理共用目錄

Adobe Commerce的B2B可讓您使用不同公司的自訂價格，維護閘道共用目錄。 除了標準主要產品目錄之外，它還可以讓客戶存取具有不同定價結構的兩種共用目錄。

如果在設定中啟用了「共用目錄」功能，管理員仍可看見原始主要目錄，但店面中只會顯示「預設（一般）」公用共用目錄。 此外，可建立僅對特定公司帳戶成員可見的自訂目錄。
