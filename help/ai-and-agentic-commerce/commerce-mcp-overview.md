# Commerce MCP概覽

[!BADGE 即將推出]{type=Informative tooltip="此功能即將推出，且可能會有變動。"}

適用於Adobe Commerce的Commerce MCP可協助開發人員使用即時商務資料和工具，建置從探索到結帳的自訂代理程式體驗。 在此影片中，自訂AI助理使用Commerce MCP，透過瞭解購物者意圖和提供相關的產品推薦來將對話轉換為轉換。

## 這部影片是給誰看的？

* Commerce商店擁有者
* 銷售商
* 數位行銷人員

## 視訊內容

* 使用自然語言（而非篩選器和搜尋列）探索對話式產品
* 內容感知工作階段管理，保留購物車內容和購物者偏好設定
* 透過輸入進行雙向購物車管理，不需要重新載入頁面或確認模式
* 結帳已在交談中完成，未導覽到其他位置
* 特定購買產品的購買後內容指引
