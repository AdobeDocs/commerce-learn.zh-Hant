# 使用Edge Delivery提升店面效能

瞭解由Edge Delivery Services提供支援的Adobe Commerce店面如何取得卓越的速度和彈性。 瞭解智慧型頁面呈現、持續快取和CDN架構如何立即改善Lighthouse分數、Core Web Vitals和SEO。

## 這部影片是給誰看的？

* 前端開發人員
* 網站績效工程師
* Ecommerce Manager

## 視訊內容

* 瞭解Edge Delivery架構如何讓店面在Lighthouse中達到90以上的分數並傳遞Core Web Vitals。
* 探索CDN支援的傳送和快取如何在不增加工作量的情況下改善SEO、流量和轉換率。
