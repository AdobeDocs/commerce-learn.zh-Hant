# 站點範圍分析工具控制板

站點範圍分析工具是一種主動的自助服務工具和中央儲存庫，包括詳細的系統見解和建議，以確保Adobe Commerce安裝的安全性和可操作性。 它提供全天候即時效能監控、報告和建議，以確定潛在問題。 使用此工具可以更好地瞭解站點運行狀況、安全性和應用程式配置，幫助縮短解析時間，並提高站點穩定性和效能。
