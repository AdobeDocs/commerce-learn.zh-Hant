# 使用即時搜索

{{ee-feature}}

來自Adobe Commerce的Live Search為Adobe Commerce提供快速、超相關、直觀的搜索體驗，無需額外付費。 由Adobe Sensei提供支援的Live Search使用人工智慧和機器學習算法對聚集的訪問者資料進行深入分析。 此資料與您的Adobe Commerce目錄結合後，將帶來高度吸引力、相關性和個性化的購物體驗。
