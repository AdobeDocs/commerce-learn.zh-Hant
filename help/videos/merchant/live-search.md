# 使用Live Search

{{ee-feature}}

來自Adobe Commerce的即時搜尋為Adobe Commerce提供快速、超相關且直覺式的搜尋體驗，不需額外付費。 Adobe Sensei支援的即時搜尋採用人工智慧和機器學習演算法，對匯總的訪客資料執行深入分析。 此資料與Adobe Commerce目錄結合後，可產生高度吸引人、相關且個人化的購物體驗。
