# 設定和偵錯

本影片是Adobe Commerce Cloud技術入門系列的一部分。 它說明YAML設定檔案如何跨環境運作、哪些工具可協助您驗證和最佳化這些檔案，以及您如何進行偵錯和記錄。 您也會看到ECE-Tools和Quality Patches Tool如何配合、New Relic如何彙總記錄、如何使用Xdebug進行遠端偵錯，以及如何跨多個應用程式伺服器管理記錄。

## 這部影片是給誰看的？

* DevOps工程師
* Commerce架構師
* 後端開發人員

## 視訊內容

* 跨雲端環境的YAML設定和環境特定的設定。
* 驗證、最佳化和偵錯工作流程。
* ECE-Tools、品質修補工具、New Relic、Xdebug和記錄管理。
