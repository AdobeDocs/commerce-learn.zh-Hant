# 技術入門 — 設定和偵錯

瞭解Adobe Commerce Cloud中用於不同環境設定的關鍵Yaml檔案、可用於驗證和最佳化這些設定的工具，以及偵錯和記錄程式。 瞭解環境特定設定的重要性、如何使用EC工具和品質修補程式，以及New Relic在記錄彙總中的作用。 深入瞭解使用Xdebug進行遠端偵錯，以及跨多個應用程式伺服器管理記錄檔的必要性。
