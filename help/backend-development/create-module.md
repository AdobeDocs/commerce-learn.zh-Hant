# 建立模組

模組是 [!DNL Commerce]  — 整個系統都建立在模組之上。 通常，建立自定義項的第一步是構建模組。

## 這段錄像是給誰的？

- 開發人員

## 添加模組的步驟

- 建立模組資料夾。
- 建立etc/module.xml檔案。
- 建立registration.php檔案。
- 運行bin/magento設定。
- 升級指令碼以安裝新模組。
- 檢查模組是否工作正常。
