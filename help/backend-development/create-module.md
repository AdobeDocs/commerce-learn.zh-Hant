# 建立模組

模組是 [!DNL Commerce]  — 整個系統是以模組為基礎構建的。 通常，建立自訂的第一步就是建立模組。

## 這段錄像是給誰的？

## 新增模組的步驟

- 建立模組資料夾。
- 建立etc/module.xml檔案。
- 建立registration.php檔案。
- 執行bin/magento設定。
- 升級指令碼以安裝新模組。
- 檢查模組是否正常運作。
