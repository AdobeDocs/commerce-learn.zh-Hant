# 建立新頁面

{{only-for-on-prem-commerce-cloud}}

建立一個傳回json的頁面，並包含一個引數。

## 這部影片是給誰看的？

- 開發人員

## 新增頁面的步驟

- 建立模組
- 新增&#x200B;**routes.xml**&#x200B;檔案
- 新增控制器（動作）檔案

## 建立模組的步驟

- 建立模組資料夾
- 建立`etc/module.xml`檔案
- 建立`registration.php`檔案
- 執行`bin/magento setup:upgrade`指令碼以安裝新模組
- 檢查模組是否正常運作
