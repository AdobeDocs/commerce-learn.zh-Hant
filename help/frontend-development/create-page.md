# 建立新頁面

使用一個參數建立可傳回json的頁面。

## 這段錄像是給誰的？

- 開發人員

## 新增頁面的步驟

- 建立模組
- 新增 **routes.xml** 檔案
- 添加控制器（操作）檔案

## 建立模組的步驟

- 建立模組資料夾
- 建立 `etc/module.xml` 檔案
- 建立 `registration.php` 檔案
- 執行 `bin/magento setup:upgrade` 安裝新模組的指令碼
- 檢查模組是否正常運作
