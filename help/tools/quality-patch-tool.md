# 使用品質修補工具

品質修補工具是命令列工具，可為Adobe Commerce和Magento Open Source提供高品質的修補程式。 它可讓您：

- 檢視最新品質修補程式的一般資訊
- 套用品質修補程式
- 還原已套用的修補程式

Adobe Commerce支援團隊和Magento Open Source社群會開發品質修補程式。
