# 使用質量修補工具

質量修補程式工具是一個命令行工具，可為Adobe Commerce和Magento Open Source提供高質量修補程式。 它允許您：

- 查看有關最新質量修補程式的一般資訊
- 應用質量修補程式
- 還原已應用的修補程式

Adobe Commerce支援團隊和Magento Open Source社區開發質量補丁。

這段錄像是給誰的？

- 後端開發人員
- 站點管理器
