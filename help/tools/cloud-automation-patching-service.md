# Cloud Automation修補服務

Adobe的Cloud Automation Patching Service (CAPS)是Adobe Commerce全網站分析工具(SWAT)中的一項功能，可簡化探索和應用安全性修補程式的程式。 使用者可透過SWAT控制面板下的Commerce管理員存取CAPS，選取其專案和環境，並檢視可用的修補程式。

## 這部影片是給誰看的？

* 網站管理員
* 開發人員
* DevOps和系統管理員
