# 全網站分析工具儀表板

全網站分析工具是主動式自助服務工具和中央存放庫，其中包含詳細的系統分析和建議，以確保Adobe Commerce安裝的安全性和可操作性。 它提供全天候的即時效能監控、報告，以及識別潛在問題的建議。 使用此工具可更清楚地瞭解網站健康狀況、安全性和應用程式設定，有助於縮短解決時間，並改善網站穩定性和效能。
