# 使用PhpStorm中的升級相容性工具

{{only-for-on-prem-commerce-cloud}}

* 升級相容性工具的功用
* 如何將其與Magento PhpStorm外掛程式搭配使用
* 如何依模組嚴重性和不相容性計數讀取HTML結果
