# 在PHP Storm上使用升級相容性工具

升級相容性工具(UCT)是免費工具，可在幾分鐘內分析目前版本和目標升級版本之間的不相容性。 PHP Storm外掛程式可讓此工具更易於使用。

在本影片中：

- 什麼是升級相容性工具？
- 如何將其與MagentoPHP Storm外掛程式一起使用
- 如何存取HTML結果，以協助依嚴重性和不相容性數目將模組視覺化
