# 可觀察性示範

觀看Adobe Commerce中的即時可觀察性示範。 此示範會示範Webhook和事件工作流程如何連線不同的服務，以便更輕鬆追蹤訂單和解決問題。 檢視訂購方式、錯誤修正方式及處理檢查方式。  瞭解同步和非同步流程。 本檔案也會說明追蹤和記錄如何出現在Grafana和New Relic中以進行疑難排解。 瞭解關聯對於尋找慢點和維持系統順暢運作很重要的原因。
