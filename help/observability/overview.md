# 可觀察性概觀

瞭解集中式可觀察性為何對現代Adobe Commerce架構至關重要。 隨著擴充性發展成為具有獨立應用程式、Webhook和UI擴充功能的可撰寫模型，商家面臨著追蹤複雜執行流程的挑戰。 瞭解統一的可觀察性層如何關聯各服務的訊號，以啟用主動式最佳化並確保快速、可靠的結帳體驗。

## 視訊內容

* 可撰寫模型增加複雜性：獨立應用程式和Webhook缺乏單一內容。
* 集中可觀察性是關鍵：關聯所有元件的記錄檔和量度。
* 提升結帳效能：快速找出瓶頸，確保購物順暢。
