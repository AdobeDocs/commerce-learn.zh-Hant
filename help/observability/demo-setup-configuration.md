# 設定與組態

瞭解如何使用標準化、相關的遙測技術，實現跨Adobe Commerce和自訂應用程式的端對端可觀察性。 此影片說明為何關聯和標準化很重要，介紹Commerce可觀察性模組和IO即時遙測程式庫。 瞭解這些工具如何啟用統一的監控、記錄及追蹤，以實現順暢的操作流程。

## 這部影片是給誰看的？

* Adobe Commerce開發人員
* 解決方案架構師
* 技術產品經理

## 視訊內容

* 標準化並關聯遙測，以實現完全可觀察性
* 針對自動化訊號使用Commerce可觀察性模組
* 使用應用程式的IO Live Telemetry Library延伸監視
