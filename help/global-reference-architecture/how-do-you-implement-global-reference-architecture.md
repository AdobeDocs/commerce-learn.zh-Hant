# 實作全球參考架構

瞭解如何實作全球參考架構(GRA)。 瞭解有助於設定成功GRA的早期決策。 取得如何區隔網站的想法、成功的關鍵，以及以有條理方式開發功能的一些概念。

## 客群

* 需要瞭解此概念的人。
* 具有多個品牌的大型專案

## 視訊內容

* 早期決策是成功GRA的基礎
* 在GRA中發佈網站的想法
* 成功的全球參考架構專案的關鍵
* 命名慣例
* 為基本功能集選擇正確的品牌或網站
* 主要角色和責任
* 全域控制板 — 使用哪些功能
* 使用GRA作為模組市場的替代概念
