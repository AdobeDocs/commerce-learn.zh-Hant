# SQL REPORT BUILDER

Adobe Commerce Intelligence SQL Report Builder可讓您查詢資料倉儲、檢閱結果，並將其轉換為圖表。 當您使用SQL建立報表時，不會等待更新週期來調整您建立的欄。 如果系統有所偏差，請編輯並重新執行查詢，直到輸出符合您的預期為止。

>[!IMPORTANT]
>
> 您需要管理員許可權才能使用SQL Report Builder。

>[!TIP]
>
> 如果您是Commerce Intelligence的新手，[Adobe Commerce Intelligence使用手冊](https://experienceleague.adobe.com/docs/commerce-business-intelligence/mbi/guide-overview.html){target="_blank"}將說明如何彙總、分析和視覺化資料，以獲得更強大的業務見解。

* 業務使用者
* 資料分析師

## 視訊內容

* 開啟SQL Report Builder並對您的資料倉儲執行查詢。
* 從查詢結果建立視覺效果。
* 將報告儲存至您團隊的報告面板。
