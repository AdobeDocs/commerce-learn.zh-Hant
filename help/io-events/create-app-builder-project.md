# Create an App Builder project

Learn how to create an Adobe Developer App Builder project to consume events from Adobe Commerce. 在[安裝適用於Adobe Commerce的Adobe I/O Events](https://developer.adobe.com/commerce/events/get-started/installation/){target="_blank"}找到其他檔案。

## 這部影片是給誰看的？

* 剛開始使用I/O事件的Adobe Commerce和Adobe Developer App Builder的開發人員，需要建立Adobe App Builder專案。

## 視訊內容 {#video-content}

* Creating an App Builder project
* Using Adobe Developer Console
* Using Adobe I/O

## 有用的命令 {#useful-commands}

```bash
mkdir myproject && cd myproject

aio login

aio app init

aio app run
```
