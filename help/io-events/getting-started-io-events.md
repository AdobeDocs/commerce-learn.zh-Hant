# 開始使用Adobe Commerce的IO事件

## 這部影片是給誰看的？

* 剛開始使用I/O事件之Adobe Commerce和Adobe Developer App Builder的開發人員，必須建立AdobeApp Builder專案。

* 瞭解IO事件
* 將事件傳送至Amazon Event Bridge的可能性
* 瞭解如何設定和設定Adobe主控台
* 基本Adobe I/O命令
* 從Adobe Commerce傳送事件至Slack頻道
