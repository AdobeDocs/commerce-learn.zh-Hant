# 建立產品屬性

Adobe Commerce使用屬性來管理與產品、客戶和產品評等相關聯的資訊。 屬性是可顯示為表單、報告、產品評論、分層導覽和電子郵件訊息欄位的資訊片段。 它們是產品目錄的建置組塊，並描述產品的特定特性。
