# [!DNL Page Builder] 概覽

[!DNL Page Builder]可讓您使用自訂版面輕鬆建立內容豐富的頁面，以增強您的視覺storytelling，並提高客戶參與度和忠誠度。

從2.4.3版開始，[!DNL Page Builder]預設可在Adobe Commerce和Magento Open Source中使用。 透過Adobe Commerce中的[!DNL Page Builder]，您可以合併測試和預覽、個人化內容和產品推薦，以及即將推出的其他功能。

在本影片中，瞭解如何：

* 編輯現有頁面
* 編輯新頁面
* 將產品新增至現有頁面
* 新增及格式化列和欄
* 將媒體和元素新增至欄和列
* 新增滑桿並設定其格式
