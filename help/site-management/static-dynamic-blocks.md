# 建立靜態和動態區塊

區塊是內容的模組化單位，可放置在頁面上最顯眼的位置。 內容區塊有時稱為靜態區塊或CMS區塊。 它們可用來顯示固定資訊，例如文字、影像和內嵌視訊，以及由Widget提供或源自資料庫或其他來源的動態資訊。 首頁上的大多數元素都是可以輕鬆管理的區塊。 在本影片中，瞭解如何建立靜態和動態區塊。
