# 管理SEO的URL重寫

「URL重寫」工具可讓您變更與產品、類別或CMS頁面相關聯的任何URL。 重新寫入生效時，任何指向先前URL的連結都會重新導向至新位址。

- 網站管理員
- 電子商務商品商
- 電子商務行銷人員
